## Run Locally  

Clone the project  

~~~bash  
git clone https://github.com/Tiketku/BE-Tiketku.git
~~~

Go to the project directory  

~~~bash  
cd BE-Tiketku
~~~

Install dependencies  

~~~bash  
npm install
~~~

Setup env

Setup Sequelize

~~~bash  
npx sequelize db:create
npx sequelize db:migrate
npx sequelize db:seed:all
~~~

Start the server  

~~~bash  
npm run start
~~~
